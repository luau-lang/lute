# Lute Lint

`lute lint` is a programmable linter for Luau code, invoked by calling `lute lint <LUAU FILE>` (use `lute lint --help` for more uses). It is _programmable_, meaning that you can write a new lint rule for your Luau code _in Luau_. In addition, it is built on top of the official Luau language stack, meaning that it uses the same parser that Roblox does, unlike 3rd party linters like Selene, which use custom parser implementations. `lute lint` also comes with a set of default lint rules which will run automatically when it is called. These default lint rules are under active development, with the eventual goal of matching Selene's functionality. Also in progress is [Mandolin](https://github.com/luau-lang/mandolin), a VSCode extension to surface lint violations in the editor, rather than just on the command line. Both `lute lint` and Mandolin are works in progress, so if you run into any problems or have any questions, feel free to reach out to skanosue@roblox.com or the rest of the Luau team!
